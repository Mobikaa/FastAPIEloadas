# FastAPI Eloadas

A simple FastAPI application for managing items and users.

## Features

- User management
- Item management
- RESTful API endpoints

## Installation

1. Clone the repository
2. Install dependencies: `pip install fastapi uvicorn`
3. Run the server: `uvicorn main:app --reload`

## API Endpoints

- `/users` - User operations
- `/items` - Item operations

## Usage

Start the server and visit `http://localhost:8000/docs` for interactive API documentation.
